
BASE INSTRUCTIONS

-------------------------------------------------------------------------------------------------------------------------------------
The agent.homes is the main dataset that only contains one row for each object.
Use also data from the other tables if needed.
The references tables are aggregated data per municipality, postal codes and grunnkrets (Basic Statistical Unit). 
 Specifically for Oslo City, we have  districts. Focus on the aggregated data within the smallest grouping first, that is grunnkrets, moving outwords to postal_codes and at last to municipality.
The help tables gives different values for location and demographic data.
